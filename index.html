Rilan Tech

F.A.Q

Pertanyaan Yang Sering Diajukan

Apakah ketika Laptop/PC saya di instal ulang OS nya terhapus datanya?

Installasi OS biasanya akan menggunakan Local Disk C (Data C), oleh karena itu Kami dalam melakukan instal ulang OS pada sebuah Laptop/PC biasanya melakukan backup atau mengamankan terlebih dahulu data penting di Data C ke Local Disk Lain, jadi datanya aman tidak akan terhapus.

Apakah ketika Laptop/PC saya di instal ulang OS nya, aplikasi nya terhapus?

Secara umum ketika aplikasi di instal pada sebuah Laptop/PC, file hasil installasinya akan tersimpan di Local Disk C (Data C), jadi otomatis setelah Kami instal ulang Laptop/PC Anda, semua aplikasi akan terhapus. Tetapi tenang, Kami selalu menginstal kembali aplikasi yang umum anda gunakan.

Apakah Laptop/PC saya perlu diinstalkan aplikasi anti virus?

Tergantung versi Windows Laptop/PC Anda, jika anda menggunakan Windows 7 atau versi dibawahnya biasanya harus diinstalkan anti virus, tetapi jika Laptop/PC Anda Windows 8/10/11 maka tidak perlu diinstalkan aplikasi anti virus karena Windows versi tersebut telah memiliki aplikasi anti virus bawaan.

Apakah Laptop/PC atau Printer saya dapat diselesaikan servicenya hari ini juga?

Tergantung tingkat kerusakan perangkat Anda, jika kerusakan ringan bisa diselesaikan hari ini juga, jika kerusakan berat paling cepat 1 minggu dan paling

lambat 1 bulan.

Laptop saya ketika di cas tidak bisa mengisi bateray nya, kira-kira apa penyebabnya ya?

Biasanya ketika laptop tidak mengisi daya pada bateray nya ketika di cas disebabkan oleh bateray nya yang sudah rusak/kalah, jadi solusinya harus diganti bateray nya dengan bateray baru.
